# Atlantic BRS

Welcome to the Atlantic BRS github. There are several things here:

- public
  - analysis tools
  - data and code repositories of published papers
  - other data and presentations

- private:
  - repositories for data processing, on going analysis, report generation.
  - links to data repositories and other internal documents
  - internal documentation

## The data archive

### Take me to the archive:

The [AtlanticBRS  main archive](https://duke.box.com/s/i5x7wk5s7ai2s317fto3qbnpgvw5a02w) is on box. If you need access or are having difficulty with the box, please send an email to wrc14 [at] duke [dot] edu.

Some quick links to commonly used resources:

- [ATLANTIC-BRS_CEE Metadata](https://duke.box.com/s/vtiawx7xznhporib8s1zjlef3ymu14on). Detailed file with information on all experiments.
- [CEE Metadata flat csv](https://duke.box.com/s/mi9ravh1nvkq04xjcq4pnx476epeulj1). Flat csv with the basic information about each CEE.
- [Processed sattag datastreams](https://duke.box.com/s/26yxi6zxnoz801vt0vze4x9ynj0b1rxd). There is some more documentation in this [repo](https://github.com/atlanticbrs/DATAPREP_sattag_processing). The [raw sattag data](https://duke.box.com/s/0xq7irgf8w0h3dr10rn9gv8gp5avia49) is still available as well.
- [Sattag deployment metadata](https://duke.box.com/s/26yxi6zxnoz801vt0vze4x9ynj0b1rxd). Some basic information on all deployed sattags.
- [Sattag CEE](https://duke.box.com/s/7rv41oxnm3t01lnilh3jr8dr15p7c599) dive plots for each experiment (all tags that were actively collecting dive data). 
- [Sattag gonio gps match](https://duke.box.com/s/puxd3shhmaz6hffywynv5n6xxgf2s4hu).  Some processed goniometer positional 'ancillary' data.
